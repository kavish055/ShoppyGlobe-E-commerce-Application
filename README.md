## Project Overview

ShoppyGlobe enables users to browse products, view detailed information, manage their cart, and complete a checkout process. Built for learning and portfolio, it uses latest React patterns, robust state management, routing, and performance optimizations.

---

## Features

- **Product Listing:** Fetches products from an API and displays them in a responsive grid with search filtering.
- **Product Details:** Shows full details for any product in an animated modal/page.
- **Cart Management:** Users can add, update quantities, remove items, and view cart totals.
- **Checkout:** Simple form collects user data with live order summary; places order with confirmation and auto-redirect.
- **Redux/Context State:** Cart and search state management using Redux-like reducers with React Context.
- **Dynamic Routing:** Custom router implementation using hash routing to support all navigation.
- **Error Handling:** Graceful error messages for data fetch failures and route mismatches.
- **Responsive Design:** Works beautifully across desktop, tablet, and mobile.
- **Animations:** Smooth fade, slide, spin, and bounce animations for engaging user experience.
- **Performance:** Implements code splitting, lazy loading for images and components, optimizing initial load.
- **ESLint Integration:** Enforced code quality and style rules for maintainability.
- **Accessibility:** Basic accessibility for keyboard and screen reader users.

## Installation and Setup

1. **Clone the Repository**
git clone-
https://github.com/kavish055/ShoppyGlobe-E-commerce-Application.git

cd ShoppyGlobe-E-commerce-Application

2. **Install Dependencies**
npm install


3. **Run Development Server**
npm run dev


Access at [http://localhost:5173]
(http://localhost:5173) (or whichever port Vite uses).

4. **Build for Production**
npm run build  


5. **Preview Production Build**
npm run preview
